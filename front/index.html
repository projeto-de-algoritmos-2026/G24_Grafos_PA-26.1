Regras/Como usar

O Ri Robos é uma ferramenta/site inspirada no jogo de tabuleiro Ricochet Robots



que permite o usuário visualizar o número mínimo de jogadas/movimentos necessárias para que o robô chegue até sua base de cor correspondente (objetivo) selecionada.

É possível movimentar os robôs utilizando seu mouse!, criando combinações personalizadas.
